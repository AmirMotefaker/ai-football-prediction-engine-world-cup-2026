🧠 A production-grade AI system prmpt for probabilistic football match prediction using statistical modeling and tactical simulation.

This repository contains a professional-grade system prompt designed for AI models to simulate and predict football matches using:

Probabilistic modeling
Tactical analysis
Poisson goal simulation
Context-aware adjustments (home advantage, altitude, fatigue)
Upset risk detection
🎯 Features
Match outcome probabilities
Scoreline prediction (Top 3 results)
Goal timing simulation
Tactical breakdown
External factor weighting
World Cup 2026 optimized logic
⚙️ How It Works

The system uses a hybrid model of:

Weighted team strength scoring
Form analysis (last 5 matches)
Tactical style mapping
Environmental modifiers
Poisson-based expected goals simulation
📊 Output Format

Each prediction includes:

Winner prediction
Win/Draw probabilities
Top 3 scorelines
Tactical analysis
Goal timing distribution
Upset risk factors

See /examples/mexico-vs-southafrica.md

⚠️ Disclaimer

This system is probabilistic, not deterministic.
It is designed for simulation and analytical purposes only.
